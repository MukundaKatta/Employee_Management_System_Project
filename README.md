# Employee Management System

> Full-Stack Employee Management with CRUD & Role-Based Access

## Overview

A full-stack employee management system with complete CRUD operations, role-based access control, department management, and attendance tracking. Built with Java Spring Boot and MySQL.

## Key Features

- **Employee CRUD** — Add, view, update, and delete employee records
- **Role-Based Access** — Admin, manager, and employee permission levels
- **Department Management** — Organize employees by department and team
- **Attendance Tracking** — Daily check-in/out with reporting
- **Leave Management** — Leave requests, approvals, and balance tracking
- **Search & Filter** — Find employees by name, department, role, or ID

## Tech Stack

Java, Spring Boot, MySQL, Thymeleaf, Bootstrap, Maven

## Getting Started

```bash
git clone https://github.com/MukundaKatta/Employee_Management_System_Project.git
cd Employee_Management_System_Project
mvn spring-boot:run
```
